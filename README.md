# StepStyle

StepStyle is a complete and responsive shoes e-commerce platform designed for customers to browse, search, purchase, and manage footwear products online. 
It also provides an admin panel for managing products, inventory, customers, orders, categories, brands, and discount coupons.
The project is built using PHP, MySQL, HTML, CSS, JavaScript, AJAX, PDO, and an MVC-style architecture.

## Main Features

Customer Features

* Secure user registration and login
* Browse available footwear products
* Search products by name, brand, or category
* Filter and sort products
* View complete product information
* Add products to cart
* Manage product quantities in the cart
* Add or remove products from the wishlist
* Apply available discount coupons
* Place orders through the checkout system
* View order history and order details
* Cancel eligible orders
* Manage profile and account settings
* Switch between dark and light themes

Admin Features

* Secure and separate admin authentication
* Dashboard with store statistics
* Add, edit, view, and delete products
* Manage product images, prices, stock, and status
* Manage product categories and brands
* Monitor available and low-stock products
* Update inventory manually
* View inventory activity logs
* Manage customer orders
* Update order delivery status
* Restore stock for cancelled, returned, or refunded orders
* View registered customers and their order history
* Create and manage discount coupons
* Use dark and light admin themes

## Key Characteristics

* Modern and user-friendly interface
* Fully responsive design
* Separate customer and administrator panels
* Database-driven content and account management
* Real-time cart updates using AJAX
* Secure database operations using PDO
* Role-based access control
* Input validation and output escaping
* CSRF protection for sensitive actions
* Password hashing for secure authentication
* Inventory tracking and stock management
* Transaction-based order processing
* Dark and light mode support

## Project Purpose

StepStyle is suitable for:

* Online shoe and footwear stores
* Small and medium footwear businesses
* Product, inventory, and order management
* Learning PHP and MySQL development
* Demonstrating full-stack development skills
* E-commerce portfolio and academic projects

## Technologies Used

Frontend: HTML5, CSS3, Vanilla JavaScript, AJAX, Font Awesome
Backend: PHP 8+, PDO
Database: MySQL / MariaDB
Development Tools: XAMPP, phpMyAdmin, Visual Studio Code, Git, GitHub

## Security

StepStyle includes prepared SQL statements, password hashing, session-based authentication, CSRF protection, role-based authorization, input validation, output escaping, secure file handling, and database transactions.
